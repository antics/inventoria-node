Redis Key Structure
===================

    | Namespace:key    | Data              | Type    |
    |------------------|-------------------|---------|
    |                  | <uid>             |         |
	| i:<item_id>      | <image_id>        | Hash    |
    |                  | <info>            |         |
    |                  | <title>           |         |
    |------------------|-------------------|---------|
    |                  | <email>           |         |
    | u:<uid>          | <title>           | Hash    |
    |                  | <info>            |         |
    |------------------|-------------------|---------|
    | e:<email_addr>   | <uid>             | String  |
    |------------------|-------------------|---------|
    | d:<word||u:<uid> | [item_id_1..id_n] | Set     |
    |------------------|-------------------|---------|
    | s:<session_id>   | <session_data>    | Various |
    |------------------|-------------------|---------|
    | count:uid        | [1..2..n]         | Set     |
    |------------------|-------------------|---------|    	
    | count:items      | [1..2..n]         | Set     |
	|------------------|-------------------|---------|
    | items            | [item_id(n)]      | ZSet    |
    |------------------------------------------------|
